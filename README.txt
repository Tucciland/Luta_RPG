	O projeto consiste em um script interativo desenvolvido em JavaScript, HTML e CSS, utilizando uma abordagem orientada a objetos. Trata-se de um jogo de RPG simplificado, no qual os usuários podem escolher entre uma variedade de combatentes e monstros para participarem de batalhas.
	Este script permite aos jogadores visualizarem as imagens representativas de cada personagem, realizar ataques e receber feedback instantâneo sobre a eficácia dos golpes, tudo isso apresentado de forma intuitiva por meio de elementos visuais. Além disso, o jogo inclui uma área de registro (LOG), onde os eventos e resultados das batalhas são registrados para acompanhamento posterior.
